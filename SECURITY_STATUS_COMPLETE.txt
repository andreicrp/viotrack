   Attack: Modified/fake QR code
   Result: Token validation fails, access denied

   Attack: id=1' OR '1'='1
   Result: Prepared statements prevent injection

8. Unauthorized Role Access
   Attack: Teacher tries to edit violations
   Result: Button hidden + API validation fails
